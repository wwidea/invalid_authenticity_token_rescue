## Installation
Add this line to your application's Gemfile:

```ruby
gem 'invalid_authenticity_token_rescue'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install invalid_authenticity_token_rescue
```
